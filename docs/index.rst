General Introduction
====================

This section covers the general concepts of OMERO. Those concepts are demonstrated using
the Web client. This section introduces the data management functionalities of OMERO, shows how to annotate data and demonstrates how to search for data and metadata. Also, the management of groups and users is presented, both in the Web client as well as on the command line.
